# 🚀 API Testing: Conduit Project

To repozytorium zawiera zautomatyzowane testy API dla aplikacji [Conduit](https://conduit.mate.academy/api/). Projekt został przygotowany w celu weryfikacji funkcjonalności backendu.

## 🛠 Technologie
* **Postman** – do budowania kolekcji zapytań.
* **JavaScript** – do pisania testów (Chai assertions).

## 🧪 Co jest testowane?
Moja kolekcja zawiera testy dla:
* **Autoryzacji:** Rejestracja i logowanie użytkownika.
* **Artykułów:** Tworzenie, edycja i usuwanie postów.
* **Profilu:** Zarządzanie danymi użytkownika.
* **Komentarzy:** Dodawanie i usuwanie komentarzy pod artykułami.

## 📝 Przykłady testów
W moich skryptach wykorzystałam:
* **Pre-request scripts:** Automatyczne przygotowanie danych testowych.
* **Test scripts:** Walidacja kodów odpowiedzi (status 200, 201) oraz weryfikacja schematu JSON.
